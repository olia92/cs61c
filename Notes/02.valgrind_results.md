## Έλεγχος του προγράμματος με το [Valgrind](https://valgrind.org/)

#### Εκτέλεση με nvc
```
make clean
make benchmark
./benchmark benchmark

```
αποτέλεσμα:

```
RUNNING BENCHMARK ON 1200 PICTURES...
Making network...
Loading batches...
Loading input batch 0...
Segmentation fault (core dumped)

```


# Valgrind

```
valgrind --leak-check=full --show-leak-kinds=all ./benchmark benchmark
```
